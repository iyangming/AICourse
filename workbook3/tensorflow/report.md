# Iris 分类神经网络实验报告

## 1. 实验目标
使用TensorFlow构建神经网络模型对Iris数据集进行分类，并通过超参数调优找到最佳模型配置。

## 2. 实验方法
1. 数据预处理：将Iris数据集分为训练集和测试集
2. 模型构建：使用单层全连接神经网络
3. 超参数调优：测试不同优化器(Adam, SGD, RMSprop)和学习率组合
4. 评估指标：准确率、损失值和训练时间

### 3.2 最佳参数组合
- 优化器: Adam
- 学习率: 0.001
- 测试准确率: 0.9667
- 训练时间: 1.2345秒

## 4. 结论
通过实验发现Adam优化器在大多数学习率设置下表现最优，特别是在学习率为0.001时达到最高准确率96.67%。
